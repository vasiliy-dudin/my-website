## Design System

- Developed the design system, made it reusable. Made styles more user-friendly and scalable to enable adding more features to the interface.
- Created complex components for interactive tasks and personal accounts.
- Implemented a palette using the OKLCH colour model to improve accessibility and visual hierarchy of elements.
- Implemented design tokens, led the process of accelerating frontend deployments.
